Hey, I'm Tarun. 

I’m a computer science student currently spending most of my time figuring out systems programming, software development, and automation. Right now, I'm focused on mastering **Java**, learning **Rust**, and writing **Bash** scripts to automate the boring stuff.

### What I write about
Instead of letting my project notes, configuration tweaks, and debugging fixes sit in random text files on my computer, I put them here. I mostly write about:

*   **Systems & Scripting:** What I'm learning while diving into Rust and breaking things with Bash.
*   **Linux & Setup:** Managing my dotfiles, fixing desktop configurations, and customizing my environment.
*   **Java & Logic:** Figuring out how to build clean, efficient software without making the code a complete mess.
*   **Project Logs:** Honest write-ups of things I'm building, what broke, and how I actually fixed it.

### Why this blog exists
I started this site because the best way to make sure I actually understand a concept is to try and explain it to someone else. Whether you're another student or just looking for a specific configuration fix, I hope you find something helpful here.

### Get in touch
If you want to chat about code, configurations, or work on a project together, feel free to reach out.
